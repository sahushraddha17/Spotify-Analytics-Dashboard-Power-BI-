# Spotify Analytics Dashboard (Power BI)

## Project Overview
This project is an interactive Spotify Analytics Dashboard built using Power BI. 
The dashboard analyzes Spotify music data to provide insights into songs, artists, 
and music trends through interactive visualizations.

## Objectives
- Analyze Spotify streaming data
- Identify popular artists and tracks
- Understand music trends using data visualization
- Present insights through an interactive dashboard

## Tools & Technologies
- Power BI
- Power Query
- DAX
- Data Visualization

## Features
- Interactive dashboard with multiple visuals
- Filters and slicers for better analysis
- Smooth navigation using buttons
- Data modeling and DAX calculations
- Clear storytelling through charts and graphs

## Insights
- Top artists and most streamed tracks
- Trends in music popularity
- Comparison of different songs and artists
